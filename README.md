# 📚 ExamGhost

> Turn any study material (PDFs, notes, slides, textbook pages) into a smart, timed exam simulation system that tests your understanding like a real exam.

---

## 🚀 What is ExamGhost?

ExamGhost is an AI-powered exam practice tool that transforms uploaded learning materials into interactive multiple-choice exams.

Instead of passively reading notes, users actively test themselves under real exam conditions with:

- Timed exams
- MCQ questions generated from their own materials
- Voice-based interaction (read + answer)
- Performance analytics across all past attempts

---

## 🎯 Problem It Solves

Most students:
- read notes without testing themselves
- don’t know what they actually understand
- struggle to simulate real exam conditions

ExamGhost fixes this by turning any content into a **real exam experience instantly**.

---

## ⚙️ How It Works

1. Upload study material (PDF, notes, slides, etc.)
2. Select exam settings:
   - Number of questions
   - Difficulty level
   - Time duration
3. System generates MCQ questions from the content
4. Take a timed exam one question at a time
5. Get instant results + analytics

---

## 🧠 Key Features

- 📄 Upload any learning material
- 🤖 Automatic MCQ generation
- ⏱️ Timed exam mode (real exam simulation)
- 🔊 Voice reads questions aloud
- 🎤 Voice answer support
- 📊 Performance analytics dashboard
- 📈 Progress tracking over multiple exams

---

## 🧪 Tech Stack

- Next.js (full-stack app)
- Browser Speech APIs (voice input/output)
- AI API
- SQLite (local data storage)

---

## 🧠 What Makes It Different?

Unlike normal quiz apps, ExamGhost:
- simulates real exam pressure
- tracks performance over time
- builds a personal learning history
- adapts to user behavior through analytics
- can interact with voice
